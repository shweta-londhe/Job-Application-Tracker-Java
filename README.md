# Job Application Tracker (Java + MySQL)

## 📌 Overview
This is a Java-based console application using JDBC to connect with MySQL database. It allows users to add and view job applications.

## 🚀 Features
- Add job application
- View all applications
- Menu-driven interface
- Real-time database integration

## 🛠️ Technologies Used
- Java
- JDBC
- MySQL

## ▶️ How to Run
1. Setup MySQL database `job_tracker`
2. Update DB credentials in DBConnection.java
3. Run Main.java
